# Quiz Application 🎯

A simple and responsive Quiz Application built using **HTML, CSS, and JavaScript**.  
It includes a login page, multiple-choice questions, score calculation, and result display with percentage and pass/fail status.


## 🚀 Features

- Login page (enter username)
- 10 multiple-choice questions
- Instant answer feedback
- Score calculation
- Percentage result
- Pass/Fail system
- Play again option
- Responsive design
- 

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
4. Start the quiz!
